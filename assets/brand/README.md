# Brand assets

Canonical logo files for nlqdb.

- `logo.svg` - full logo with icon and wordmark; prefer this for web and docs.
- `icon.svg` - icon-only mark; use this for favicons and square avatars.
- `logo.png` - square PNG export for services that do not accept SVG.
- `icon.png` - square PNG icon export for services that do not accept SVG.

When the logo changes, replace files in this directory first, then follow
[`BRAND.md`](../../BRAND.md).
